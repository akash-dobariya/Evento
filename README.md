# EVENTO Showcase Website

This project has been converted to a basic HTML, CSS, and JS website.

## Structure

- `index.html`: The main entry point containing all HTML structure and embedded CSS/JS configuration.
- `assets/`: Contains images and other static assets.

## Usage

Simply open `index.html` in any web browser. No build steps or Node.js required.

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Lucide Icons (via CDN)

## Backup

The original React source code has been moved to the `_react_backup/` directory.
